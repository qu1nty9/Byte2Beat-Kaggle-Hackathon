# Data Notes

Raw data files are not intended to be blindly committed to GitHub.

## Included Source Documents

Competition PDFs, DOCX files, and the project idea catalog can be tracked when file size and permissions allow.

## Raw Datasets

The provided datasets live locally under:

```text
docs/drive-download-20260627T215828Z-3-001/Datasets/
```

This path is ignored by `.gitignore` because it includes large raw data, especially the ECG CSV. For Kaggle, these should be attached as Kaggle resources or referenced through the official competition resource folder.

## Derived Outputs

Small generated tables and figures under `outputs/` are tracked when useful for review and writeup development.
